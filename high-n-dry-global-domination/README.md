# High N' Dry: Global Domination

An idle/incremental game about growing High N' Dry from a single restaurant into a global
chain — open new locations, automate the kitchen, hire staff, and expand across cities and
countries.

This folder is a brand new, standalone project. It does not share code, assets, or history
with anything else in this repo.

## Status

Scaffolding only — design and build have not started yet.

## Structure

- `assets/` — art, audio, and other game assets
- `src/` — game source (once we pick a build approach)

## Next steps

Design details still need to be nailed down before building starts, e.g.:

- Target platform (browser single-file HTML like the other games here, native app, etc.)
- Core idle loop (what generates currency, what you spend it on, prestige/reset mechanic?)
- Progression theme (menu items → restaurants → cities → countries → planets?)
- Art style
- Build approach: a full spec-driven build (like `unicorn-academy/`) or a lighter prototype
  (like `unicorn-reading-game/`)
